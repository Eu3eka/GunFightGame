8.设计四种敌人，每种敌人的运动逻辑不一样：
	第一种携带手枪，只会在给定的平台上范围中来回巡逻，当玩家与他高度差进入一定范围无论是否面向玩家均立刻面向玩家并立即开始射击，最多五次，
		最多5次停顿一会；停顿结束重复刚才的射击。
	第二种携带步枪，会在全屏范围内移动，会进行跳跃，如果与玩家同向且进入警戒高度差则立即开始射击，连续射击最多10次后停顿，结束后重复刚才的射击。
	第三种携带散弹枪，会直接追击玩家，靠近玩家一定距离时直接射击一次。
	第四种携带狙击枪，会与玩家保持一定距离，会进行跳跃，如果与玩家同向则射击两次。
	“最多”指敌人弹夹中子弹数量多余该数目。若少于则打出所有子弹后立即换弹。换弹后不进行射击。
	一旦敌人进入射击状态就会完成所有射击次数，除非弹夹打空。
	每一个敌人都有射击间隔（值在constans.h中已有定义），间隔减为0才能下一次射击。
	射击过程中如果弹夹打光，停止射击并进行换弹。
